== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Slider Hero with Elementor'
3. Activate Slider Hero with Elementor from your Plugins page.

=== Manually ===

1. Upload the `slider-hero-with-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the Slider Hero with Elementor plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
